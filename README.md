# 真探堡健独家 V2.0 · FitX Scorer

一个面向健身房探店、场馆横评和品牌巡店的本地化评分工具。

FitX Scorer 试图回答一个很具体的问题：

> 一家健身房到底好不好，能不能不只靠感觉？

它把器械配置、自由力量区、空间环境、服务体验、周边配套和加分/扣分项拆成可记录、可计算、可复核的评分流程，让一次探店不只停留在主观印象，而是沉淀成可以导出、归档和横向比较的数据。

## 在线使用

如果 GitHub Pages 已启用，可直接访问：

```text
https://henry-sdqd-lab.github.io/fitx-scorer/
```

也可以下载仓库后直接打开：

```text
index.html
```

如浏览器对本地文件下载或脚本权限有限，建议用本地服务打开：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://127.0.0.1:8080/
```

## 核心能力

- **多维度评分**：覆盖固定器械、自由器械、有氧器械、空间布局、更衣、服务、周边配套和加分/扣分项。
- **移动端友好**：单页 HTML 应用，适合现场用手机边看边记。
- **面积基准自动计算**：支持训练区面积输入；未知训练面积时可按总面积 80% 折算；超过 2500 平米的基准仍按 2500 平米计算。
- **器械逻辑细分**：支持插片/挂片比例、部位覆盖、专项杠铃、深蹲架、卧推台、可调凳、硬拉台等规则化评分。
- **结果可复核**：每个关键分项都会在备注中保留原始选择、数量基准和计算结果。
- **Excel 原表导出**：自动写入馆名、城市、总分、分项得分和备注，适合归档和二次整理。
- **末页 PNG 海报**：导出包含总分、评级、雷达图百分比和分项指标的分享图。
- **本地优先**：无后端、无账号系统，数据主要在浏览器本地处理。

## 适用场景

- 健身房探店评分
- 城市场馆横向测评
- 连锁品牌巡店检查
- 私教、教练或训练者选馆记录
- 场馆改造前后对比
- 自媒体内容的数据化支撑

## 评分维度

### 器械区

- 固定器械覆盖度
- 插片 / 挂片比例
- 肌群覆盖与多角度训练能力
- 自由力量配置
- 专项杠铃、深蹲架、卧推台、硬拉台等力量区细项
- 有氧设备数量、类型和状态

### 环境

- 空间布局与动线
- 功能训练区
- 体感、通风、卫生
- 地面、镜面、灯光、视听体验
- 更衣、淋浴和卫生间

### 服务与周边

- 门禁、体测机、AED
- 人员服务与维护秩序
- 教练、团课和自由教练友好度
- 补给、开放时间、卡价合理度
- 交通、停车、生活配套和辐射人群

### 加分 / 扣分

- 恢复设施、泳池、造型室、Hyrox / CrossFit 等加分项
- 场地脏乱无人管理、抽烟及电子烟管理无效等扣分项

## 导入与导出

- 首页支持导入原表 Excel，用于继续修改历史评分。
- 末页支持导出评分 JSON、导出原表 Excel、导出末页 PNG。
- JSON 更适合作为备份；对外提交或归档建议使用 Excel 和 PNG。

## 项目定位

本工具不是为了制造唯一标准，而是为了提供一个更透明的起点：

让用户知道分数从哪里来，让场馆知道差距在哪里，也让好器械、好配置、好场馆被更多人看见。
